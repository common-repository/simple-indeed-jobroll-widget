Simple Indeed Jobroll Widget

== Description ==
This is a simple widget to show Indeed Jobroll widget on a publisher site. It's a simple widget that can earn money for you. 
Please read about [Indeed Publisher](https://ads.indeed.com/jobroll/).
You can see a [Live Demo](http://rezaur.net). 
Please let me know if you have any issue regarding this Plugin at [rezaur.net](http://rezaur.net/contact).
I Will be very happy if this Plugin helps anyway. Reviews will inspire me to add new features and more customize able. 

== Installation ==
1. Upload the `simple-indeed-jobroll-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget to your widget area from `Appearance > Widgets`
1. Add the Indeed Publisher Id and hit save, Done.
